## 시작 방법

```
git clone https://github.com/lodado/boilerplate

cd boilerplate

npm install

npm run build (webpack dev build 시작)

npm start (express 시작)

```

윈도우라면 위 npm 2 명령어 대신(build, start)

```
npm run test
```
로 동시에 시작 가능
